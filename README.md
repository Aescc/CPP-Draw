# CPP-Draw
### Create instances of circles by clicking and dragging in a way that appears as the desired image or symbol.
#### Key Binds
* Mouse on corner = Resize canvas.
* L-Bracket = Shrink brush size.
* R-Bracket = Enlarge brush size.
